Image Placeholder Instructions

Note: This file is meant to help you set up the images for your portfolio. You can delete this file after setting up your images.

Required Images

Your portfolio requires the following images to be placed in the /images directory:

Filename Size (recommended) Usage
profile.jpg 400 x 400px Your profile picture in the About section
hero-bg.jpg 1920 x 1080px Background image for the hero section
project1.jpg 800 x 600px Project 1 thumbnail
project2.jpg 800 x 600px Project 2 thumbnail
project3.jpg 800 x 600px Project 3 thumbnail
project4.jpg 800 x 600px Project 4 thumbnail

Where to Get Images

You can use your own images or get free placeholder images from these sources:

  • Unsplash - Free high-quality stock photos
  • Pexels - Free stock photos and videos
  • Placeholder.com - Simple image placeholders
  • Lorem Picsum - The Lorem Ipsum for photos

How to Use

  1. Download or create images according to the recommended sizes above
  2. Rename them to match the filenames listed in the table
  3. Place all images in the /images directory of your portfolio
  4. If you want to use different filenames or add more images, make sure to update the references in the HTML code

Additional Tips

  • For better website performance, optimize your images before uploading them
  • You can use tools like TinyPNG or Squoosh to compress images without losing quality
  • Consider using modern formats like WebP for even better performance, but provide JPG/PNG fallbacks for better browser compatibility
  • Make sure you have the rights to use any images you include in your portfolio
